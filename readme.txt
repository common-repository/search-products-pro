== Description ==
PRO Search Bar for any WooCommerce website! Search Products PRO is integrated with popular page builders and themes.

Elementor and WPBakery Page Builder widgets and elements are included.

= Ultimate search options! =
This search bar will always find products for any keyword input. Customers will love it.

* Search by Product SKU
* Search by Categories
* Search by Attributes
* Search by Tags
* Search by Colors
* Search by Material
* Search by Brands
* Search by Custom Taxonomies
* Search by Product Meta
* Search by Price
* Search by ACF fields
* Call-Out Search Modal
